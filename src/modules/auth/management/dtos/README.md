# Management DTOs

DTOs for validation and structured inputs across management APIs.

Files
- `create-org.dto.ts`, `create-invitation.dto.ts`, `accept-invitation.dto.ts`
- `create-role.dto.ts`, `create-permission.dto.ts`, `set-role-permissions.dto.ts`
- `graphql-types.ts`, `register-user.input.ts`

Notes
- All DTOs use `class-validator` decorators and work with the global ValidationPipe.
